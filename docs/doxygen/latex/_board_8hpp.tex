\doxysubsection{Detailed Description}
Declares the \doxylink{class_board}{Board} class which represents a hexagonal grid of tiles. 

The \doxylink{class_board}{Board} class manages a collection of \doxylink{class_tile}{Tile} objects arranged using cube coordinates. It provides functions for tile retrieval, neighbor lookup, ownership and color assignment, and overall grid initialization.

The cube coordinate system ensures consistent hex-\/grid relationships via the constraint\+: $x + y + z = 0$.
