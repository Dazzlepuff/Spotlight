\doxysubsection{Detailed Description}
Declares the \doxylink{class_colors}{Colors} class which stores the color standards. 

Throughout the project seven standard colors and an additional eighth neutral color are commonly referencedand this class standarizes the use of the colors to allow for flexibilty if additional colors are added or removed later.
