\doxysubsection{Detailed Description}
Represents a single board tile that can be owned by a company and display a color. 

Each tile may have an owner (a pointer to a \doxylink{class_company}{Company}) and an associated color. Tiles start unowned and "{}neutral"{} in color by default.
