\doxysubsection{Detailed Description}
Declares the \doxylink{class_deck}{Deck} class used for managing collections of \doxylink{class_card}{Card} objects. 

The \doxylink{class_deck}{Deck} class represents a container of \doxylink{class_card}{Card} instances. It provides functionality for loading cards from JSON files, shuffling, drawing, transferring cards between decks, and querying deck state.

\begin{DoxyNote}{Note}
This class assumes that the \doxylink{class_card}{Card} type supports construction from JSON objects. 
\end{DoxyNote}
