\doxysubsection{Detailed Description}
Represents a hexagonal game board composed of cube-\/coordinate tiles. 

The \doxylink{class_board}{Board} is implemented as an unordered map keyed by \doxylink{struct_cube_coord}{Cube\+Coord}, enabling constant-\/time lookup of tiles. Each tile may store its owner (\doxylink{class_company}{Company}) and visual color data. The board supports neighbor detection and manipulation of tile attributes.

The design is suitable for strategy or simulation games utilizing hex grids.
