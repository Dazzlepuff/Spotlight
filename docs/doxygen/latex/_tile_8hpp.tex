\doxysubsection{Detailed Description}
Declares the \doxylink{class_tile}{Tile} class, representing a single board tile that can be owned and colored. 

Each \doxylink{class_tile}{Tile} tracks its ownership by a \doxylink{class_company}{Company} and a color state, providing methods for setting and retrieving this information. Tiles default to being unowned and "{}neutral"{} in color.
