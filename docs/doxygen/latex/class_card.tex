\doxysubsection{Detailed Description}
Represents a playable card with triggers and associated actions. 

Each card contains a name, description, and a set of triggers that define when certain actions occur (e.\+g., "{}on\+Play"{}, "{}on\+Start\+Of\+Day"{}). Actions are stored as JSON objects that can be executed dynamically.
