\doxysubsection{Detailed Description}
Represents a company, corporation, or in-\/game faction. 

The \doxylink{class_company}{Company} class stores immutable identifiers that describe an entity’s identity and branding in the simulation or game world. Its instances can be linked to tiles, assets, or scores via references or pointers.
