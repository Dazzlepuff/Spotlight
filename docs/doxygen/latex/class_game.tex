\doxysubsection{Detailed Description}
Central game controller managing all gameplay mechanics and state. 

\doxylink{class_game}{Game} acts as the orchestrator for a turn-\/based hex board strategy game. It maintains\+:
\begin{DoxyItemize}
\item \doxylink{class_game}{Game} state (current day, active player)
\item \doxylink{class_player}{Player} roster and their associated companies
\item Hex board and tile ownership
\item \doxylink{class_card}{Card} decks (draw and discard)
\item Resource economy and transactions
\item Command-\/based interface for game actions
\item Rendering and event handling via SFML
\end{DoxyItemize}

The class follows a command pattern for user interactions, mapping string commands to handler functions that manipulate game state. This design allows for scripting, replay systems, and network play extensions.

\begin{DoxyNote}{Note}
The game uses a -\/1 convention for player\+Index to indicate "{}current active player"{} 
\end{DoxyNote}
